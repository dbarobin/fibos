1. create fibos account
1.1 open `createAccount.js`
1.2 choose an account name
1.3 run `node createAccount.js`
1.4 it should run successfully, copy the private key
1.5 paste the account name and private key into config.json (replace "account" and "privateKey")
1.6 account is created on fibos mainnet

2. transfer from eos account to fiboscouncil
2.1 compele your eos account and private key in config.json
2.2 open `transfer.js`, modify the amount
3.3 run `node transfer.js`

3. check balance on fibos
3.1 run `node balance.js`
3.2 it should show eos balance just transferred in

4. exchange eos to fibos
4.1 change the eos amount
4.2 run `node exchange.js`
4.3 check balance again
